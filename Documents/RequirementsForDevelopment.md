# Requirements For Development

## VSCode
You need the extentions below (at least what we used and preferred):

###### SHOULD HAVES
*   C/C++ Extention Pack by Microsoft
*   C/C++ Include Guard by Akira Miyakoda
*   CMake Tools by Microsoft
*   CMake by twxs
*   koroFileHeader by OBKoro1
*   VsCode Action Buttons by Seun LanLege
*   Dart by Dart Code   (For UI)
*   Flutter by Dart Code    (For UI)
You can pick any relevant extention that does the same job, does not really matter

###### USEFUL ONES
*   BUGD Bookmark Sidebar Project by Bugd

## Git
We used Gitlab for our class and Github for our real open source version control.
